# Mini-Project-C-Language
The Inventory Management System is a console-based application written in the C programming language that allows users to manage inventory items efficiently. The system provides features to add, display, search, update, and delete items from the inventory. It is designed using structures, arrays, and functions to demonstrate file organization and data management in C.

This project is useful for beginners learning C programming, especially topics like:

Structures
Functions
Arrays
Input/Output handling
Data validation
Features
Add new items to inventory
Display all inventory items
Search item by ID
Update item quantity and price
Delete item from inventory
Input validation to prevent incorrect data entry
Prevents duplicate Item IDs
Simple menu-driven interface
Technologies Used
Programming Language: C
Compiler: GCC / MinGW
Platform: Windows / Linux
Program Structure

The program uses a structure to store item details:

Item ID
Item Name
Quantity
Price

All records are stored in an array and managed using different functions such as:

addItem()
displayItems()
searchItem()
updateItem()
deleteItem()
